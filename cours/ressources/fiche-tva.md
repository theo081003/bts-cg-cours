## Taux de TVA en France (2026)

| Taux | Application |
|------|-------------|
| **20%** | Taux normal (biens & services courants) |
| **10%** | Taux intermédiaire (restauration, travaux) |
| **5,5%** | Taux réduit (produits alimentaires, livres) |
| **2,1%** | Taux super-réduit (médicaments remboursés, presse) |

---

## Calculs Essentiels

```
HT → TTC :  Prix TTC = Prix HT × (1 + taux)
TTC → HT :  Prix HT  = Prix TTC / (1 + taux)
TVA       = Prix HT × taux
```

**Exemple taux 20% :**
- HT = 1 000 € → TTC = 1 200 €
- TTC = 1 200 € → HT = 1 000 €
- TVA = 200 €

## Déclaration TVA
