## 6. Notes et remarques

### 6.1 Orthographe et coherance
- Le dossier s'appelle `cours/comptabilite` (sans accent sur le 'e' final) mais le breadcrumb affiche parfois "comptabilité" avec accent
- Tous les fichiers utilisent l'orthographe sans accent dans les noms (ex: `01-bases-comptable.md`)
- Le contenu utilise les accents pour une meilleure lisibilite

### 6.2 Structure des fichiers
- Tous les fichiers suivent le meme format :
  - Metadonnees YAML (frontmatter)
  - Titre H1
  - Sections H2, H3
  - Tableaux pour les ecritures comptables
  - Section "A Retenir"
  - Section QCM
