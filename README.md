## 📝 Comment ajouter un cours

1. Crée un fichier `.md` dans le bon dossier (ex: `cours/comptabilite/02-bilan.md`)
2. Utilise le [modèle disponible](./cours/ressources/TEMPLATE-cours.md)
3. Ajoute une ligne dans [`COURSES.md`](./COURSES.md)
